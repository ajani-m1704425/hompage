About our library

Anima 95 allows you add animations to elements on your project without having to get your hands dirty with codes, bringing your projects to life with awesome animations.

Our library gives you unlimited access to a variety of awesome animation effects and it basically comprises a simple, lightweight CSS Library built using SAS.

Anima 95 helps you save time and speed up your project development game. As browsers don't understand raw SASS files like they understand CSS, linking them to your HTML file like you would normally do with your CSS file won't yield results.

So we compiled our SASS files into a CSS file for the animations to work in any browser of your choice, using a task runner called Gulp and the installation of node.js - node package manager used to install Gulp and some plugins into our project.
